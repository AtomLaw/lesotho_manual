The following tutorials were prepared for use by the students of the 3rd Fingal County Council Planner training course in Maseru, Lesotho. Each of these makes extensive use of data created by #MapLesotho and the OSM community.

If you are following these guides on windows you must install both QGIS and the OpenGeo Suite 3.x. If you are using linux you should install PostgreSQL, PgAdmin3, PostGIS and QGIS in order to follow these guides.
